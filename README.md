#SOAP-SERVICE
- a SOAP (simple object access protocol) message is used to connect the client to the web service using XML format.
- WSDL (web serice description language) provides a contract to the consumer
- UDDI (universal description discovery language) is a registary for consumers to query and get access to the WS
- WS (Web service) allows communication of 2 different machines on 2 different servers to communicate.
